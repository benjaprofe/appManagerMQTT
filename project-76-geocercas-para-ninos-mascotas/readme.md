# Geocercas para Niños Mascotas

## Categoría: Maps

Alerta inmediata si el tracker GPS sale de una zona segura (parque/casa).

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
