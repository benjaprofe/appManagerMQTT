# Semáforo Inteligente Adaptativo

## Categoría: Smart City

Control de tráfico que cambia según la densidad de autos detectada.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
