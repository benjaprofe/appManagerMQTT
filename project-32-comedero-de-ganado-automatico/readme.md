# Comedero de Ganado Automático

## Categoría: Agriculture

Dispensa alimento según el RFID del animal y registra su consumo.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
