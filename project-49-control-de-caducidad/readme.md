# Control de Caducidad

## Categoría: Retail

Sistema que alerta descuentos en productos próximos a vencer en estantería.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
