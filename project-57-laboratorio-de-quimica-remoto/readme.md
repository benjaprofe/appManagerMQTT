# Laboratorio de Química Remoto

## Categoría: Entertainment

Brazo robótico que mezcla líquidos peligrosos controlado por alumnos.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
