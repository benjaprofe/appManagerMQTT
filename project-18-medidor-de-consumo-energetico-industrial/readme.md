# Medidor de Consumo Energético Industrial

## Categoría: Industrial

Dashboard que muestra el gasto eléctrico por máquina en tiempo real.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
