# Mapa de Densidad de Transporte

## Categoría: Maps

Visualización en vivo de dónde están todos los buses/metros de la ciudad.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
