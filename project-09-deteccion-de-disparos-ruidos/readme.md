# Detección de Disparos Ruidos

## Categoría: Smart City

Triangulación de sonidos fuertes para alertar a policía en zonas peligrosas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
