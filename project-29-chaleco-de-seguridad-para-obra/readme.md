# Chaleco de Seguridad para Obra

## Categoría: Healthcare

Detecta caídas, golpes de calor y ritmo cardíaco de trabajadores.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
