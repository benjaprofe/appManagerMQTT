# Detector de Inhibidores de Señal

## Categoría: Security

Alerta si detecta intentos de bloquear la señal WiFi/GSM de la alarma.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
