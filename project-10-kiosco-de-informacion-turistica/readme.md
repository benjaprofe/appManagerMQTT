# Kiosco de Información Turística

## Categoría: Smart City

Pantalla interactiva que sugiere rutas y envía mapas al móvil del turista.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
