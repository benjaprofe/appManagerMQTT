# Logística de Última Milla

## Categoría: Maps

Mapa para el cliente que muestra exactamente dónde está su repartidor en bici.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
