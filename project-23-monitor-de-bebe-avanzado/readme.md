# Monitor de Bebé Avanzado

## Categoría: Healthcare

Analiza el llanto y monitorea respiración/temperatura, alertando al móvil.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
