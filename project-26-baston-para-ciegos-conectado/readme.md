# Bastón para Ciegos Conectado

## Categoría: Healthcare

Detecta obstáculos con ultrasonido y avisa a cuidadores botón de pánico.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
