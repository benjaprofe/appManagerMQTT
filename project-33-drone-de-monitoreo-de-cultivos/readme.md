# Drone de Monitoreo de Cultivos

## Categoría: Agriculture

Estación base que recibe datos de vuelo y estado de plantas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
