# Feedback de Cliente Físico

## Categoría: Retail

Botones de satisfacción en salida que alertan al gerente si hay muchas quejas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
