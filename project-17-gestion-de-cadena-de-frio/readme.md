# Gestión de Cadena de Frío

## Categoría: Industrial

Sensores en camiones frigoríficos que alertan si la temperatura sube durante el transporte.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
