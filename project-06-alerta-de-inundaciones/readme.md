# Alerta de Inundaciones

## Categoría: Smart City

Sensores en alcantarillas y ríos que activan sirenas y barreras automáticas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
