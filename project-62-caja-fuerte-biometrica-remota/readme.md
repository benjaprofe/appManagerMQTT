# Caja Fuerte Biométrica Remota

## Categoría: Security

Solo se abre si el usuario pone su huella Y aprueba desde la app móvil.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
