# Rastreo de Flota de Peligros

## Categoría: Maps

Mapa en tiempo real de camiones con carga peligrosa y alertas de desvío.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
