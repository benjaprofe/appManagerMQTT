# Refrigerador de Vacunas

## Categoría: Healthcare

Control estricto de temperatura con batería de respaldo y alertas críticas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
