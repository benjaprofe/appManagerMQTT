# Rastreo de Activos en Almacén

## Categoría: Industrial

Localización precisa de montacargas y pallets dentro de una nave industrial.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
