# Simulador de Carreras

## Categoría: Entertainment

Volante y pedales que envían telemetría a un dashboard de ingeniero de pista.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
