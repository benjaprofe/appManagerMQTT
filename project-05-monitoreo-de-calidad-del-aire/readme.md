# Monitoreo de Calidad del Aire

## Categoría: Smart City

Estaciones en paradas de bus que miden CO2 y recomiendan uso de mascarilla.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
