# Sistema Andon Digital

## Categoría: Industrial

Botoneras en líneas de montaje para alertar problemas y detener la producción.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
