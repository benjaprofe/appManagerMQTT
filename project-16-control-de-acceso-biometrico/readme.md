# Control de Acceso Biométrico

## Categoría: Industrial

Puertas de seguridad que requieren huella/RFID y registran entradas en tiempo real.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
