# Rastreo de Ganado GPS

## Categoría: Agriculture

Collares que definen un "cerco virtual" y alertan si un animal sale de la zona.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
