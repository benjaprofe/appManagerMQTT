# Monitor de Colmenas

## Categoría: Agriculture

Mide peso (producción de miel) y zumbido (salud de la reina) en apiarios.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
