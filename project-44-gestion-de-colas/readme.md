# Gestión de Colas

## Categoría: Retail

Sensores que abren nuevas cajas automáticamente cuando la fila supera X personas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
