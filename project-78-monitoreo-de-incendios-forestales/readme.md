# Monitoreo de Incendios Forestales

## Categoría: Maps

Mapa que actualiza el avance del fuego según sensores de temperatura en el bosque.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
