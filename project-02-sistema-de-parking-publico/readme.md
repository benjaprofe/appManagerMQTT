# Sistema de Parking Público

## Categoría: Smart City

Sensores en plazas de calle que guían a los conductores a lugares libres vía app.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
