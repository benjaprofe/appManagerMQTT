# Sistema de Llamada a Enfermería

## Categoría: Healthcare

Botones en habitaciones que priorizan urgencias en el smartwatch de enfermeras.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
