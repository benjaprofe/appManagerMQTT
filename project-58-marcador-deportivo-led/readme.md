# Marcador Deportivo LED

## Categoría: Entertainment

Controlado desde móvil para partidos de barrio, con bocina y tiempo.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
