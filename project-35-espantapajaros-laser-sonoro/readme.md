# Espantapájaros Láser Sonoro

## Categoría: Agriculture

Detecta movimiento de aves y activa disuasivos sin dañar a los animales.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
