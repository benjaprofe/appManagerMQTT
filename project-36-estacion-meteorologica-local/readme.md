# Estación Meteorológica Local

## Categoría: Agriculture

Mide viento, lluvia y UV para automatizar toldos y riegos de la zona.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
