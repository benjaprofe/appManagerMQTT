# Cafetería Automatizada

## Categoría: Retail

El cliente pide desde la mesa (QR) y la máquina prepara el café en orden.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
