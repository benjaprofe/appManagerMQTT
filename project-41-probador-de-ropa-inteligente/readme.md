# Probador de Ropa Inteligente

## Categoría: Retail

Espejo que detecta la prenda (RFID) y permite pedir otra talla desde la pantalla.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
