# Control de Aforo en Salas de Espera

## Categoría: Healthcare

Conteo de personas para mantener distancia social y ventilación.

## Componentes
- Dashboard de Monitoreo
- Panel de Control

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
