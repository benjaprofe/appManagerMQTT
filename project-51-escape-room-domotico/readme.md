# Escape Room Domótico

## Categoría: Entertainment

Sala donde las pistas activan mecanismos reales (abrir cajones, luces) vía MQTT.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
