# Parada de Bus Inteligente

## Categoría: Smart City

Pantalla que muestra ubicación real del bus y permite solicitar parada para discapacitados.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
