# Monitor de Vibración de Maquinaria

## Categoría: Industrial

Detecta patrones anómalos en motores para predecir fallos (Mantenimiento Predictivo).

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
