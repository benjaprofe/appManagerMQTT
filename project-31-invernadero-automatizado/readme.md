# Invernadero Automatizado

## Categoría: Agriculture

Control total de ventanas, ventiladores, riego y luces UV según el clima.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
