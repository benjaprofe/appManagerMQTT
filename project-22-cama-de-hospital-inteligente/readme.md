# Cama de Hospital Inteligente

## Categoría: Healthcare

Detecta si el paciente se levanta o si hay humedad (incontinencia).

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
