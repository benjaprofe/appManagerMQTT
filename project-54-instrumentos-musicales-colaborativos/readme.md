# Instrumentos Musicales Colaborativos

## Categoría: Entertainment

Varios usuarios tocan desde sus casas y suena en una sala central.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
