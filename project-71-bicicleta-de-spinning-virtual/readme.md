# Bicicleta de Spinning Virtual

## Categoría: Gym

La resistencia de la bici cambia según la pendiente de la ruta en la pantalla.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
