# Gestión de Residuos Urbanos

## Categoría: Smart City

Contenedores que avisan a los camiones cuando están llenos para optimizar rutas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
