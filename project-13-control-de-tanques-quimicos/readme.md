# Control de Tanques Químicos

## Categoría: Industrial

Monitoreo de nivel, presión y temperatura con válvulas de alivio automáticas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
