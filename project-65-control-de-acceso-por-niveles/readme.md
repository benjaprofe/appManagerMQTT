# Control de Acceso por Niveles

## Categoría: Security

Tarjetas RFID que dan acceso a diferentes zonas según el rango del empleado.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
