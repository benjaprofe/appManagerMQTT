# Inodoro Inteligente

## Categoría: Healthcare

Sensores básicos que analizan frecuencia y tiempo de uso para salud digestiva.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
