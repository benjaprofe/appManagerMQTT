# Riego de Parques Públicos

## Categoría: Smart City

Sistema que riega solo si la humedad del suelo es baja y no hay previsión de lluvia.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
