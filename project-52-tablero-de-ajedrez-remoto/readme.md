# Tablero de Ajedrez Remoto

## Categoría: Entertainment

Mueves piezas físicas y se mueven en el tablero del oponente en otro país.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
