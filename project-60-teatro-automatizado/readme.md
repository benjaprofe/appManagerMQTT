# Teatro Automatizado

## Categoría: Entertainment

Control de telón, luces y efectos de humo sincronizados con el guion.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
