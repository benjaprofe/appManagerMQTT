# Rastreo de Activos Hospitalarios

## Categoría: Maps

Mapa indoor para localizar dónde está el desfibrilador o la silla de ruedas más cercana.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
