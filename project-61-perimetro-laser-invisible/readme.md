# Perímetro Láser Invisible

## Categoría: Security

Barreras láser que al cortarse activan cámaras y luces estroboscópicas.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
