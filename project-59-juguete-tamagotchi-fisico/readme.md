# Juguete Tamagotchi Físico

## Categoría: Entertainment

Mascota robot que "vive" en un servidor y interactúas con ella físicamente.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
