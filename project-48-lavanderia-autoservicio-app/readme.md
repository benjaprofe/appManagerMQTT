# Lavandería Autoservicio App

## Categoría: Retail

Muestra qué máquinas están libres y avisa cuando tu ropa está lista.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
