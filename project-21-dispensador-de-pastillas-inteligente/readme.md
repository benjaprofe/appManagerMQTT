# Dispensador de Pastillas Inteligente

## Categoría: Healthcare

Libera dosis a horas exactas y notifica a familiares si no se retira.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
