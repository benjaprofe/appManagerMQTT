# Control de Heladas

## Categoría: Agriculture

Activa aspersores o calefactores en frutales cuando la temperatura baja de 0°C.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
