# Simulador de Presencia

## Categoría: Security

Enciende luces y TV aleatoriamente cuando la casa está en "Modo Vacaciones".

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
