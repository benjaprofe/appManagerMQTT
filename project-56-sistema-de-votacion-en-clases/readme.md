# Sistema de Votación en Clases

## Categoría: Entertainment

Botoneras físicas para alumnos para responder quiz en tiempo real.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
