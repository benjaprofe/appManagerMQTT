# Gestión de Silvicultura

## Categoría: Agriculture

Sensores de humo en bosques remotos para detección temprana de incendios.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
