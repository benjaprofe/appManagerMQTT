# Botón de Pánico Vecinal

## Categoría: Security

Sistema comunitario donde al pulsar un botón físico, alerta a todos los vecinos en un mapa.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
