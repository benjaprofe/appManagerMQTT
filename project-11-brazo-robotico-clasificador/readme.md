# Brazo Robótico Clasificador

## Categoría: Industrial

Cinta transportadora que separa objetos por color usando sensores RGB.

## Componentes
- Dashboard de Monitoreo
- Panel de Control
- Conexión MQTT

## Cómo usar
1. Abrir `dashboard.html` para ver el estado.
2. Abrir `control.html` para interactuar.
